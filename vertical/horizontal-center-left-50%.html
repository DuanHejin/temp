设置position为relative/absolute;
设置固定width;
设置left: calc(50% - 元素width/2);
或者left: 50%; transform: translateX(-50%); 这样就不用知道元素的width，实用性强 

            .horizontal-center {
                position: relative;
                /* position: absolute; */
                width: 400px;
                left: calc(50% - 200px);
                /* left: 50%;
                transform: translateX(-50%); */
            }
