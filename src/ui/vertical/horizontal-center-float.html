设置float:left;
设置固定width;
设置margin-left: calc(50% - 元素width/2);
或者margin-left: 50%; transform: translateX(-50%); 这样就不用知道元素的width，实用性强 

            .horizontal-center {
                float: left;
                width: 400px;
                margin-left: calc(50% - 200px);
                /* left: 50%;
                transform: translateX(-50%); */
            }
